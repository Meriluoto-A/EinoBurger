🍔 **EinoBurger**

EinoBurger is a fictional fast‑food chain used as a strategic analytics case.
The focus is on evaluating whether a shift toward plant‑based products is economically and operationally viable.

❓ Core Question
+ Should the company shift toward plant‑based products, and what does it mean for demand, pricing, costs, and operations?

📁 Data

+ Store‑level sales data
+ Product mix (meat vs plant‑based)
+ Customer satisfaction and survey data
+ Labour and fixed cost structure

🔍 Key Analyses

+ Demand modelling for plant‑based burgers
+ Survey data integration → demand uplift assumptions
+ Price elasticity and substitution effects
+ Scenario comparison: Current model, Hybrid model, Fully plant‑based model
+ Labour satisfaction vs productivity proxy
+ Cost–volume–profit analysis


🧠 Methods
+ Regression models 
+ Scenario modelling with assumptions table
+ Optional Monte‑Carlo style uncertainty ranges
+ Basic optimisation and profitability logic

📊 Outputs

+ Python: demand forecasts and scenario comparison charts
+ Power BI: strategic scenario dashboard
